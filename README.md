ESP32 Wi-Fi HTTP Server Example

This project demonstrates Wi-Fi connectivity and HTTP server hosting on ESP32 using ESP-IDF.
It serves a simple HTML page when connected to a network, showing basic web interface skills for embedded IoT projects.
